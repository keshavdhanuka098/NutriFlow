# NutriFlow AI 🥗

An AI-powered smart nutrition and wellness platform built with **React + TypeScript + Vite**. NutriFlow helps users analyze nutrition intake, generate AI-based meal plans, track hydration, visualize health data, and receive personalized fitness and wellness recommendations.

---

# 🚀 Features

* 🤖 AI-generated nutrition insights using Gemini AI
* 📊 Interactive nutrition and calorie charts
* 🍽️ Personalized meal planning system
* 💧 Daily water intake tracker
* 🔍 Smart food search functionality
* 🧠 AI wellness and fitness recommendations
* 🏋️ Personal training suggestion module
* 📞 Customer helpline integration
* 🌐 Modern responsive UI with animations
* 🎨 3D interactive experience using Three.js
* 📄 PDF/export support for reports

# ⚙️ Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/nutriflow-ai.git
cd nutriflow-ai
```

## 2️⃣ Install Dependencies

```bash
npm install
```

## 3️⃣ Configure Environment Variables

Create a `.env.local` file in the root directory:

```env
GEMINI_API_KEY=your_api_key_here
```

Get your Gemini API key from:

* [https://aistudio.google.com/app/apikey](https://aistudio.google.com/app/apikey)

## 4️⃣ Start Development Server

```bash
npm run dev
```

Application will run at:

```bash
http://localhost:3000
```
