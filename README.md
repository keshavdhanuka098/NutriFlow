# NutriFlow AI 🥗

NutriFlow AI is a nutrition and fitness web app.
It helps users track calories, water intake, and get AI-based meal suggestions.

---

# 🚀 Features

* Nutrition tracking
* AI meal suggestions
* Water tracker
* Health charts
* Responsive UI

---

# 🛠️ Built With

* React
* TypeScript
* Vite
* Tailwind CSS
* Gemini AI

---

# 📂 Project Structure

```bash
src/
├── components/
├── services/
├── types/
├── lib/
└── assets/
```

---

# ⚙️ Run Locally

```bash
git clone https://github.com/your-username/nutriflow-ai.git
cd nutriflow-ai
npm install
npm run dev
```

---

# 🔑 Environment Variable

Create a `.env.local` file:

```env
GEMINI_API_KEY=your_api_key_here
```
